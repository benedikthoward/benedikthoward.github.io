Overview

I’m building a trajectory optimizer for rocket landings using Successive Convexification (SCvx), and fine-tuneing with Gradient Descent. It handles variable-thrust gimbalable engines, gimbaled solid rocket motors (fixed thrust curve), and I’m adding active aero control surfaces (fins). The simulation is currently headless; a visualization module is in progress.

What SCvx does here

  • Linearizes the nonlinear dynamics about a reference at each iteration and solves a convex subproblem.
  • Enforces actuator envelopes (thrust magnitude, gimbal angles, angular accel limits).
  • Uses a trust region to keep each update close to the reference solution for convergence.
  • Supports control noise injection and Monte Carlo batches to evaluate robustness.

Actuator Models

Gimbalable Counter-rotating propellers motor
The simplest rocket like control model.
Gimbalable Main Engine (Variable Thrust)
Bounded thrust magnitude and gimbal angle; lateral acceleration cone enforced.
Gimbaled SRM (Fixed Thrust Curve)
Thrust curve fixed; optimizer selects gimbal to meet guidance objectives within envelopes.
Active Aerodynamics (WIP)
Fins add lateral authority at higher dynamic pressure; scheduled into the envelope.

Cost Function

Step 2 The objective encodes what a “good” landing looks like:

Terminal cost (at N):
  Jterm = wp‖pN − ptarget‖² + wv‖vN‖² + wω‖ωN‖²

Running cost (k = 0..N−1):
  Jrun = Σ‖uk‖² + λ‖uk − uk−1‖² (optional smoothness)

All terms are quadratic to keep the subproblem convex.

Constraints (Actuator-Aware)

Linearized dynamics (per step k)

xk+1 = Ak xk + Bk uk + ck

Control vector uk (depends on actuator mode)

• TVC (variable thrust): uk = [ axtvc, aytvc, az, θ, φ ]
• SRM (fixed thrust curve): uk = [ axsrm, aysrm, θ, φ ], with az(k) = ˜a;zsrm(k) (known)
• Aero fins (optional/WIP): add δ (fin deflections) to uk

TVC envelope (variable-thrust, gimbalable)

sqrt( axtvc2 + aytvc2 ) ≤ az · tan(δmax)
0 ≤ az ≤ az,max
|Δθ| ≤ ˙θmaxΔt,   |Δφ| ≤ ˙φmaxΔt

SRM envelope (gimbaled, fixed thrust curve)

az(k) = ˜a;zsrm(k)
sqrt( axsrm2 + aysrm2 ) ≤ ˜a;zsrm(k) · tan(δmaxsrm)
|Δθ| ≤ ˙θmaxΔt,   |Δφ| ≤ ˙φmaxΔt

Aero fins (linearized per iteration)

Linearize aero so fin lateral accel is affine in deflection:

alat,kfin = Dk δk (Dk depends on dynamic pressure qk, area S, CLα)
|δi,k| ≤ δi,max,   |Δδi,k| ≤ ˙δi,maxΔt

Conservative norm cap (precomputed per step):

sqrt( ax,kfin2 + ay,kfin2 ) ≤ akfin,max

Unified lateral budget (engine + fins)

ax,k = ax,keng + ax,kfin,   ay,k = ay,keng + ay,kfin

sqrt( ax,k2 + ay,k2 ) ≤ Teng,k + akfin,max
Teng,k = { TVC: az,k tan(δmax) ;   SRM: ˜a;zsrm(k) tan(δmaxsrm) }
